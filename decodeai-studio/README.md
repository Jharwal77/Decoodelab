# 🚀 DecodeAI Studio

A modern **AI SaaS Landing Page** built with **HTML5, CSS3, and Vanilla JavaScript**. This project demonstrates responsive web design, modern UI/UX principles, reusable components, animations, and interactive JavaScript features without using any frontend framework.

---

## ✨ Features

* 📱 Fully Responsive Design
* 🌙 Dark Mode with Local Storage
* 🚀 Modern Hero Section
* 💡 Features & Services Section
* 📊 Animated Statistics Counter
* 💳 Pricing Plans
* ⭐ Testimonials Section
* ❓ FAQ Accordion
* 📧 Newsletter Subscription
* 📬 Contact Form Validation
* 🔝 Back To Top Button
* 🎯 Scroll Reveal Animation
* 🔍 SEO Friendly Structure
* ♿ Accessibility Optimized

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* CSS Grid
* Flexbox
* Intersection Observer API
* Local Storage API

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Jharwal77/decodeai-studio.git
```

Move into the project:

```bash
cd decodeai-studio
```

Run using Live Server:

```bash
npx live-server
```

or

```bash
live-server
```
